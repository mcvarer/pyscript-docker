# Pyscript Simple App


### Prequests
1. Docker Engine
2. Html
3. Python


### Usage Example

1. git clone https://github.com/mcvarer/pyscript-docker.git
2. cd myapp
3. docker build -t python-html-server:v1 .
4. docker run -d -p 80:80 python-html-server:v1
5. curl http://localhost:80 (open web browser http://localhost:80)
